Значения параметра в файле конфигурации

Параметр задает стадию жизненного цикла
environment = "development"  # для разработки
environment = "testing"      # для тестирования
environment = "production"   # для продакшн-среды - работает без ограничения количества ссылок и страниц

page_count - ограничивает количество страниц для сбора ссылок (применятеся для всех стадий кроме production)

chrome_options - задает режимы, с которыми будет запущен браузер Chrome
